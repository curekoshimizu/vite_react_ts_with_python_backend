# Vite React Typescript Frontend and Python Backend

* Frontend : Vite React Typescript
 * MUI
 * React Toolkit
* Backend : Python
 * Flask

## How did I do?

1. make fe directory.

```
npm create vite
```

* input project name as `fe`
* choose `react`
* choose `react-ts`

```
cd fe
npm i
```

2. remove `assets` and `public` directories and `*.css` files.
